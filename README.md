# 基於圖論演算法之動態網格拓撲與智能尋路視覺化系統

本專案是一個基於 Python 與 Pygame 開發的動態迷宮生成與尋路演算法視覺化系統。透過實時畫面渲染，直觀展示經典圖論演算法在動態拓撲地圖中的效能與空間搜索差異。

## 🚀 核心技術與特點
* **演算法動態對評**：整合並對比非權重圖尋路（DFS, BFS）與啟發式最佳化（A*）演算法，即時顯示探索空間（Closed Set）與運算耗時（ms）。
* **非阻塞式協程機制**：利用 Python Generator 實作 Randomized DFS 迷宮生成動畫，確保流暢的畫面更新率。
* **動態拓撲路徑重算**：結合道具破壞機制（炸彈炸牆），即時更新網格連通性並觸發尋路引擎自動重新規劃路徑（Path Re-planning）。
* **響應式佈局**：支援動態矩陣維度（11x11 ~ 101x101）網格 Auto-scaling 介面自適應放大縮小。
